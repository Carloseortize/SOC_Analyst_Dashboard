# 🔐 SOC Analyst Dashboard



## 📌 Overview

Cybersecurity monitoring dashboard designed to simulate a real-world Security Operations Center (SOC).



The project analyzes logs, suspicious activities, failed login attempts, and brute-force attacks to identify security threats.



---



## 🚀 Technologies Used

- Splunk

- Kibana

- Elasticsearch

- Python

- Wireshark

- SQL

- Power BI



---



## 📊 Features

- Failed login monitoring

- Suspicious IP analysis

- Brute-force detection

- Attack timeline visualization

- Threat severity classification

- Interactive SOC dashboard
